# Password Strength Analyzer

A modern Password Strength Analyzer built using HTML, CSS, and JavaScript with a beautiful Glassmorphism UI. This application checks password strength in real time and provides suggestions to create secure passwords.

---

## Features

- Real-time password strength checking
- Glassmorphism modern UI
- Strength indicator bar
- Weak, Medium, Good, and Strong password detection
- Suggestions for improving passwords
- Responsive design

---

## Technologies Used

- HTML5
- CSS3
- JavaScript

---

## Project Structure

```bash
Password-Strength-Analyzer/
│
├── index.html
├── style.css
└── script.js
```

---

## Password Checking Rules

The application checks:

- Password length (minimum 8 characters)
- Uppercase letters
- Numbers
- Special characters

---

## How to Run

1. Download or clone the repository
2. Open the project folder
3. Run `index.html` in your browser
